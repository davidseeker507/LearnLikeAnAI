Classification picks a category. Regression predicts a value on a continuum.
Regression

Predicts: Continuous numbers (prices, temperatures, ages)

How: Learns a mathematical function mapping inputs to outputs. For example, learns that price = 200 × square_feet + 50,000 × bedrooms.

Output: A single number ($450,000, 72.5°F)

Classification

Predicts: Categories (spam/not spam, cat/dog/bird)

How: Learns boundaries that separate classes. For example, draws a line separating spam from legitimate emails based on features like word choice and punctuation.

Output: A category label, often with probabilities (85% cat, 15% dog)
